All of my projects in my 30 projects in 30 days series on Tiktok

Tiktok: @donster.coding

## How to Run

1. Install g++
2. Open terminal and cd into the project folder
3. Compile:
   g++ main.cpp -o output -Wall -std=c++17
4. Run:
   Mac/Linux: ./output
   Windows: output.exe
